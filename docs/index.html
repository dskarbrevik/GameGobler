Meet GameGobler

A cross-platform ROM library manager that makes adding and removing games on microSD cards and Android devices a seamless experience.

Download View on GitHub

Features

Everything you need to keep your retro game library organized.

📚

Browse Your Library

View your ROM collection organized by system, with cover art scraped from libretro-thumbnails and No-Intro metadata parsing.

🔌

Auto-Discover Devices

Detects USB volumes and Android devices automatically. Works with Miyoo, Anbernic, AYN, and any mass-storage device.

🚀

Fast Transfers

Copy games with real-time progress tracking. Zip extraction, space checks, and atomic writes keep things safe.

🔄

Smart Sync

Sync a curated set of games to a device — adds missing titles and removes extras in one operation.

🖼️

Cover Art

Automatically scrape box art for your entire library from the libretro-thumbnails repository.

💻

Cross-Platform

Runs natively on macOS, Linux, and Windows. Single binary — no runtime dependencies required.

Quick Start

Up and running in under a minute.

Install the app

Download the installer for your OS above. On macOS, open the .dmg and drag GameGobler to Applications. On Windows, run the installer. On Linux, use the .deb, .rpm, or .AppImage.

macOS note: The app is not yet code-signed. If you see "GameGobler is damaged", open Terminal and run:
xattr -cr /Applications/GameGobler.app

Set your library path

On first launch, a setup wizard asks you to point to your ROM directory. GameGobler expects folders named by system (e.g. Nintendo DS/, Game Boy Advance/).

Connect a device

Plug in a microSD card or Android device. GameGobler discovers it automatically and shows it in the sidebar.

Transfer games

Browse your library, pick games, and send them to your device. Track progress in real time and you're done.
